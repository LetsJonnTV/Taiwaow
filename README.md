    und funktionen hat. Um schnellst möglich ein stabiles Model

Installation:

    1. Plugins ordner in dein main verzeichnes legen
    2. Requirements installieren
        - cd plugins
        - pip install -r requirements.txt
        - cd ..
    2.5 Optional
        - README.md löschen
        - requirements.txt löschen
    3. Importieren
        - Siehe unten
